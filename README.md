# Journal d’une apprentie de Git

> *Un récit palpitant entre commandes, erreurs et illumination.*

**Jour 1.** J’ai décidé d’apprendre Git. J’ai tapé `git init`, et je crois que c’est à ce moment-là que ma vie a changé.  
**Jour 2.** J’ai fait un commit, puis un push. Git m’a dit “Everything up to date”, alors que rien ne l’était. Je soupçonne qu’il ment.  
**Jour 3.** J’ai rencontré mon premier *merge conflict*. Nous ne nous entendons pas très bien, mais il refuse de partir.  
**Jour 4.** Je crois que Git essaie de m’enseigner la patience… ou de me rendre folle.
