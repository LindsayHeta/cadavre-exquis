# Journal d’une apprentie de Git

> *Un récit palpitant entre commandes, erreurs et illumination.* 

**Jour 1.** J’ai décidé d’apprendre Git. J’ai tapé git init, et je crois que c’est à ce moment-là que ma vie a changé.  
**Jour 2.** J’ai rencontré mon premier merge conflict. Nous ne nous entendons pas très bien, mais il refuse de partir.  
**Jour 3.** Je crois que Git essaie de m’enseigner la patience… ou de me rendre folle.  
**Jour 4.** Je confirme que Git aura bien eu raison de moi… Il a eu mon cerveau.  
**Jour 5.** J’ai essayé de tout réparer avec git reset… Heureusement, j’ai pu undo, car j’ai failli effacer mon âme.  
**Jour 6.** J’ai créé plein de branches pour fuir ces problèmes.
